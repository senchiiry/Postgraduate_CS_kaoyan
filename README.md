# 一战11408计算机考研失败反思和备考经历总结
（含部分初复试资料、心得以及调剂历程等；非常的碎碎念）

## **0.写在前面：（进行一些劝退）**<br/>
这是我出分之后一直盘旋在脑内的想法：在这个考研内卷的时代，真的谨慎选择11408全家桶（除非有足够实力一定可以一志愿上岸）。考研机构如灰灰考研所说**11408调剂占绝对优势**仅仅是指调剂22、自命题学校时没有门栏，但事实上**分数上完完全全不占优**（特别是一志愿在旱区的）。<br/>
我本科中上流211，和我一起考11408成功上岸一志愿的寥寥无几，而22年**调剂系统大多直接按初试分数从上到下排列**，可想而知困难程度了。
## **0.写在前面：（进行一些反思）**<br/>
慎用b站**稍后再看**、微博**收藏**、百度网盘**保存到我的网盘**，这些囤囤鼠行为最终只会让你的这些软件中囤满了各种考研资料、考研经验，你以为存了就是会了，其实根本不是！“我先收藏一下，这份资料明天就看”这只会让你拿着的东西越来越多，而你真正掌握的却根本没有增长。
## **0.写在前面：（进行一些安利）**<br/>
我认为可以找一个考相同公共课、专业课，但不考相同学校相同专业（即没有利益冲突）的线上研友相互监督，的确能在你想偷懒的时候拉你一把。就我个人而言我不是很推荐和朋友一起备考，因为很有可能要摆就一起摆烂。当然，如果自制力足够，独自备考也没有问题

## **2.初试（330+）：**<br/>
具体准备应该是从21年3月份就开始了，寒假期间就开始做相关资料的调查，可以说我的备考过程已经算是较为充实的了。

### **数学（105+）**<br/>
所用书：**张宇高数18讲**   过了两遍，着重做了其中例题、做了每章后的习题；将其中的技巧运用补充到了笔记上；这本书其中的概念和技巧性的东西还是值得一看的，比汤家凤的网课有重点多了<br/>
**张宇概率论与数理统计9讲**   过了两遍，按这本书的脉络整理了笔记<br/>
**李永乐线性代数辅导讲义**    过了两遍，按这本书的脉络整理了笔；非常的通俗易懂，是我学的最轻松的一门<br/>
**李永乐复习全书（综合提高版）** 很早就买了，但很晚才看；主要只是看了一下知识点<br/>
**李永乐基础过关660** 做了一遍，然后重复了两遍错题；可以说这本对基础的巩固还是挺有作用的<br/>
**李永乐数学历年真题全精解析（09-21）** 真题很重要！（虽然22年我觉得数一并没有按往年题的套路来）过了两遍<br/>
**李林880+108** 最浪费的两本，也是我觉得最可惜的；题目很有难度，但题点我觉得总结得挺到位，特别是108；因为一直有点惧怕难题所以一直想着把上面那几本知识性的书多巩固几遍再开始做这两本，一直拖导致最后没时间了。**不要惧怕难题！**<br/>
<br/>
试卷：**李艳芳预测3套卷** 偏题怪题居多，不是很值得（在b站上的课很多 跟风买了）<br/>
**李永乐冲刺三套卷+六套卷+临阵磨枪** 做了下其中的填空题，难度适中；临阵磨枪没做，网上建议也是不值得买<br/>
**李林六套卷+四套卷** 全做了，题目有一定难度，其中的技巧值得一看<br/>
**余丙森五套卷** 我觉得对我来说难度最适中的一套卷子，挺不错<br/>
**张宇八套卷+四套卷** 仅做了填空，难度适中<br/>
**共创超越5套卷** 是坑，不值得买，没有答案，题目还有错的<br/>
<br/>
电子书：**真题全精解析97-08** 我是真的傻，到最后一个月才发现数学一真题还有之前的可以做，因此匆匆忙忙过了一遍，也没有深入研究（做的时候有种仅仅在检验自己能力的感觉）<br/>
<br/>
网课：**汤家凤高等数学基础班** 踩的巨坑之一，强烈建议但凡上过高数课的都不要看汤家凤的课。他网课里的知识点又多又杂，靠他讲的你根本理不出一个清晰的脉络出来，我真的觉得他讲的很不行，完全没有重点；而且这个基础班甚至连一些必要学的内容他都直接略过了（说是等强化班再讲）；我的高数笔记完全是按照这个网课整理的，这也直接导致我之后看我的笔记的时候发现既没有重点、缺漏也很多，非常的不方便<br/>
**武忠祥17堂课** 技巧值得一看！如果觉得来不及可以跟网上博主的整理，或者直接看它的课堂讲义<br/>
还零零散散听了一点张宇的强化班<br/>
个人感觉如果想听网课的话可以跟张宇或者武忠祥，但其实不建议听网课；我后期逐渐发现看网课的效率其实很低，帮助也不大，还不如直接看书
<br/>
<br/>
历程：<br/>数学大概是开始的最早的一门：具体的安排大致是3-5月网课及梳理框架→6-11月中旬强化、具体复习知识点→最后一个半月时间做卷子。<br/>
其实我的安排并不是很可取，就我个人感觉而言前几个月看的网课效果几乎为0，中间用于学习和强化的时间太长，虽然做了一定的习题，但题量并不足够（指880和108很早就买在了身边但一直没做）;后期给试卷安排的时间太少，很多卷子都只来得及做了填空题<br/>
<br/>
教训反思：**1.不能惧怕难题、难的知识点，及时练及时记忆，不要想着拖到最后**（如无穷级数这种要记的，我一直因为想着记的放最后导致这类的题全都没练，考前也只有50%的题做的出来；也不知道运气好还是怎么的今年居然全没考）<br/>
2.足够的刷题量（个人建议基础刷题量660基础+880强化+108技巧巩固），我一部分也是因为偷懒而吃了刷题量不够的亏<br/>
3.网课适当听听，不要花大量时间；4-5个月用于学习和巩固知识点，最后2个多月可以用来做卷子（也就是10月份开始，才能差不多做完且梳理明白我上面列的那些卷子）<br/>
4.计算错误一定要提前重视，这也是后期会发现让自己白白扣分却又很难纠正的地方；你会了但你没对才是最可惜的地方<br/>

### **英语（79；客观题扣8.5）**<br/>
基础水平：六级500左右（考了好几次，一直是这个水平）；对笔译有一定兴趣，过了CATTI三笔的综合能力，实务还差几分<br/>
<br/>
所用书：**王江涛写作预测20篇** 这个就是考前两天稍微看了看话题<br/>

**新文道真题狂练英二+英一（07-21）**<br/>
**（张剑黄皮书）历年考研英语真题解析及复习思路（基础试卷版+试卷版+精编版）就是01-21年的** 也就是英一的卷子完整的刷了两遍，最后分数是稳定在了50分以上；对我个人而言因为两遍间隔的时间比较长，所以并没有题目会被记住的困扰；英二就最后一个月做了三四年的阅读，太简单了每张最多错一道我就没再做了<br/>
<br/>
app：**anki（后期）**<br/>
![image](https://user-images.githubusercontent.com/48319049/163195940-f90b3a88-c342-4db6-8854-24e1c99303c7.png)<br/>
我当时差不多是按每天200的量，重复频率4天-8天的安排来的，每天差不多要1h。其实后来想想背单词花的时间的确是有点多了，好处是最后考试的时候阅读完完全全看得懂没有任何障碍，坏处是的确在复习时挤占了其它课程的时间<br/>
**不背单词（前期）** 用的2022红宝书的那个词书，完整的过了一遍<br/><br/>
电子书：**考研英语红宝书** 准确地说其实也就看了前10页左右就开始用app背单词了<br/>
**翻译真题90-19** 完整地翻译了两遍，比较缺撼的是**第一遍翻译完了并没有总结**，所以第二遍并没有很大的长进，相当于只是过了一遍<br/>
**何凯文同源外刊** 跟听力听了文章、划了不懂的词翻译并记忆了一下<br/><br/>
经验总结：1.阅读在题量上去了之后的确会有题感，我认为刷题还是可取的，后期每天尽量保持4-5篇（可以把订正反思放到第二天）<br/>
2.我不认可长难句划分学习法，**重要的还是词汇量**，划分长难句再理解有点本末倒置的感觉<br/>
3.在背单词方面我觉得用anki很可以，直接根据艾宾浩斯曲线排背诵频率（在windows和安卓上是免费的，资源可以在网上找；后文会提到）<br/>
4.练字这事情我觉得只要能写的工整其实并不一定要练；作文格式的话**段首两格段尾不空行**或者**段首不空格段尾空一行**都行，看自己的习惯<br/>
5.新题型最重要的还是能看懂，可以**利用段首段尾句提炼重点**，有一定作用<br/>
6.翻译我之前学笔译的时候有看过武峰十二天、中式英语之鉴等书，并练过三笔的往年试题，并不是很有参考价值；重要的还是**断句→划得分点并准确翻译**<br/>

### **408**（选择错了10题）<br/>
最拉跨的一门，对于科班生来说分数简直惨不忍睹，就不贴了<br/>
所用书：**天勤4本+王道4本** 天勤过了一遍，王道过了两遍；其中章末习题天勤只做了选择，王道基本全做了，且并没有好好总结，只是把答案写在了题目周围，是非常遗憾的一点<br/>
**计算机专业基础综合考纲解析+考纲（高等教育出版社出版的，因为今年考纲大改所以买了这本）** 过了一遍考纲解析，并做了其中例题；值得吐槽的是有不少错题，如果要买建议和研友一起讨论一下<br/>
**王道核心考点及模拟题** 前面的核心考点差不多是把每个章节的相对重要的内容拎出来简述了一下（因为出版的比较早，和更改后的考纲并不是那么贴合）模拟题我也主要做了选择题<br/>
**王道历年真题解析（另外打印了网上王道公布的历年真题及答案一份）** 真题选择刷了三四遍，但大题只是完整地做并梳理了一遍<br/>
**天勤最后8套模拟题** 只做了选择（相对王道模拟题来说，天勤的题没有那么贴近真题和有难度，但也有一定做的价值；如果时间足够的话，可以买来用作补充）<br/>
<br/>
时间历程：408也是从3月份左右就开始了，花了大概4-5个月的时间用于过天勤和王道的8本书，具体流程大概是看前面的知识点、划重点（做了对应知识点的知识导图），然后做课后的习题（一开始仅做了选择）；大概是9月份开始做的真题，非常遗憾的一点是我过于重视了选择题而没有花很多时间在大题上，这也是很大可能导致我最后答题时没有掌握大题正确的答题套路的原因；等考纲出了之后我大概花了一个月的时间去结合知识导图过考纲并做相关例题；最后一两个月结合真题和模拟题总结，做题。<br/>
<br/>
经验教训：1.**不要花太多时间在选择题上，而忽略了大题。大题一定要重视答题格式。** 这是我踩的大坑，希望大家可以重视<br/>
2.**如果遇到考纲大改（近几年应该不可能了），不用太担心**，因为其实最后考试时也不一定会考到多少新增的点。掌握好自己的复习节奏即可<br/>
3.**重视真题，如果做多了其实会发现每年的出题范围和方向都是有一定套路的，可以纵向横向一起比较**<br/>
4.王道和天勤4本相比较，天勤更通俗易懂，讲得比较简单；王道覆盖的知识点很全、内容很多相对比较枯燥；还是比较推荐王道，但如果非科班或者想要在某些知识点上更容易理解一点，可以买天勤<br/>

## **5.总结和其它的碎碎念：**<br/>
### a.关于时间<br/>
从正式开始到初试考完，差不多是9个月的时间。因为我的大学大三下和大四上还有很多的课程和课设，所以中间其实夹杂着很多断断续续复习的时间。<br/>
暑假我没有留校，而是在家那边的图书馆复习。早上9点开馆学到12点，下午1点学到5点半，晚上在家花2-3h学英语。<br/>
9月开始在学校图书馆租了个柜子，开始图书馆、食堂、通宵教室、宿舍四点一线的生活：上午7点40到馆，学到10点半；下午12点到6点半；晚上7点到10点半；11点开始在通宵教室学习学到凌晨1点左右回宿舍休息。<br/><br/>
### b.关于计划<br/>
我在二三月份就**做了整体的计划**，但后期并没有**根据进度及时调整并具象化**，这也是我犯的错误之一。一直想着“就根据实际情况走吧，用不着写出来并重新计划”直接导致了不断地拖延，最后书买了很多，却有很多是空白的。<br/>
我自己有**用手账本记录每天的完成情况，这是我很推荐的**。可以在一定程度上督促你自己有关任务的完成、细化每天的时间安排。<br/><br/>
### c.公众号和app推荐<br/>
时间管理：forest（种树和记录学习时间兼顾，还挺有成就感的）<br/>
单词背诵：不背单词（我试过扇贝单词、百词斩，这个是比较适合我的）<br/>
公众号：二羹师兄（anki资源）、灰灰考研&王道考研（计算机考研院校往年情况，资料是有用的、但谨慎相信主观言论）、禾拟&一烫（anki资源，有错要自己当心）、经济学人双语精读（我有空的时候就看看，提高语感）、考研斯基师兄&学长小谭考研&研途可期（各科资源都有、很全）、考研政治徐涛&腿姐考研政治课堂（后期可以跟做选择）、考研核心笔记&钦念考研（盗版资源、按需取用、非常全）、武忠祥老师（可以跟做每日一题）<br/><br/>
### d.其它<br/>
真的真的要**注意身体**，咖啡如果要喝，一定要适量。我身边脱发、失眠、得胃病的比比皆是，考研的压力只有亲身经历了才知道。我十一月份的时候几乎每天都在生理性反胃，体重狂降。<br/>
衷心劝告每个想要二战的人，如果没有对某个学校的执念，只是因为有名校情怀或者不想学历降级，一定要自己经历一遍调剂过程（指搜集可调剂院校、了解二战上岸具体的可能性、了解他人调剂经历等等）后再做决定（就我个人而言，我觉得每个深切经历过调剂过程的人大概都会对二战有点犹豫吧）<br/>
调剂的时候，多加qq和wx群，多关注发布调剂的网站和公众号，及时了解预调剂和复试信息，才不会让调剂机会白白溜走。<br/>
考研的幸存者偏差非常严重，择校一定要结合自身想法和实际情况，认真做决定。只要这个决定不会让你后悔，我觉得这个决定就没有问题。<br/>
